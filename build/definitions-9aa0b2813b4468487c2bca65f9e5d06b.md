# Definitions

- MLP: Perceptron multicapa, una red neuronal totalmente conectada usada para aproximar relaciones no lineales.
- EDA: Analisis exploratorio de datos para describir estructura, distribucion y patrones de una serie.
- ACF: Funcion de autocorrelacion que mide la dependencia entre observaciones separadas por distintos rezagos.
- PACF: Funcion de autocorrelacion parcial que cuantifica la relacion directa entre una observacion y sus rezagos controlando los rezagos intermedios.
- Rezago: Valor pasado de la serie usado como predictor de valores futuros.
- Estacionalidad: Patron que se repite con periodicidad aproximadamente constante dentro de la serie temporal.
